Web de Servicios

Somos una plataforma de uso para barrios privados cuyo objetivo es facilitar el contacto de servicios profesionales, con la particularidad de un sistema de puntaje y reseña que aumenta la confiabilidad del directorio de especialistas a los cuales recurrir

FUNCIONALIDADES/CASOS DE USO DE LA PLATAFORMA
El administrador determinará los servicios que se pueden ofrecer y otorgar roles Ir...
Los administradores pueden eliminar/censurar comentarios ofensivos, pero dejando la calificación Ir...
El proveedor puede aceptar un trabajo o cancelarlo. También puede marcarlo como finalizado una vez que el trabajo haya sido aceptado Ir...
El usuario puede pedir un servicio o cancelarlo. Una vez aceptado el trabajo puede cancelarlo o darlo por finalizado Ir...
Una vez que el trabajo fue finalizado el usuario está habilitado a calificar el servicio, calificación que se agregará al perfil del proveedor Ir...
El usuario puede contactar al proveedor, calcular un estimativo de los honorarios del proveedor por horas y calificarlo luego de prestado el servicio Ir...
